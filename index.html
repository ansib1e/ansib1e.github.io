Past Projects

Here are a few of the projects that I worked on in the past few years, click the pictures to get go to the specific project's page. For a more in-depth list of projects, please go to my Github.

FME Server Monitoring Dashboard (PowerBI + SQL Server)

Created SQL Views inside of a Micosoft SQL Server Database which housed the logs for all FME servers used by our department. These Views aggregated important data on metrics like runtimes, feature outputs, status (i.e Success, Failure, Abort), workspace properties, scheduling. This was then visualized in a Dashboard using Microsoft Power BI.

ExifEYE

Extracts exif metadata from images and audio/video files to geolocate the exact location the file was created. An example output would be the X,Y coordinates of a selfie that someone took on their iPhone 7 on vacation, or the GPS location of drone imagery/video (ex. DJI Phantom 4)

SpeedySpider

Web Scrapper that takes an item to be searched for online, like a PC Monitor, and pulls the best weekly deals off of Steam/Amazon (Canada Computers to come soon) and displays them
